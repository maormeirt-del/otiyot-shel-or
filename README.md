# אוֹתִיּוֹת שֶׁל אוֹר 🕯️
### אפליקציה שמלמדת ילדים לאהוב לקרוא עברית — עם ערכים יהודיים.

> *"כָּל יֶלֶד יָכוֹל לִקְרֹא."*
> המטרה הסופית: שילד יפתח ספר מרצונו, גם כשאין אפליקציה לידו.

---

## ✅ מה זה
אפליקציית קריאה לילדים, מכיתה א' ועד קריאה עצמאית. **אפס עלות, אפס תלות, אפס טוקנים.**
הכול רץ במכשיר — הסיפורים כתובים מראש, ההקראה והרמזים משתמשים בקול החינמי של הדפדפן,
וההקלטות של הילד נשמרות מקומית בלבד.

## 🚀 איך מריצים
**הכי פשוט — לוקאלי:**
```
cd reading-app/app
python -m http.server 5180
```
ואז לפתוח בדפדפן: http://localhost:5180

**פריסה חינם לאוויר (כמו yeshiva-site):** מעלים את תיקיית `app/` ל-Vercel / GitHub Pages. זהו. בלי שרת, בלי DB.

**כאפליקציה בטלפון:** נכנסים לכתובת בכרום בנייד → "הוסף למסך הבית". עובד אופליין.

## 🧩 מה יש בפנים
- **בחירת בן / בת** + שם → כל העברית מותאמת מגדר.
- **נֵרִי** — הדמות המלווה (להבה). התגמול = **אוֹר** שאוספים.
- **6 עולמות** (מסע הקריאה): אותיות → ניקוד → מילים קצרות → מילים גדולות → סיפורים → קריאה בלי ניקוד. **דילוג חופשי** — אפשר לקפוץ לכל שלב, עם סימון "מומלץ" לשלב הבא.
- **16 סיפורים מנוקדים** מקוריים, כל אחד עם מידה (חסד, אמת, שבת, צדקה, כיבוד הורים, ביקור חולים, שמירת הלשון, ענווה…).
- **הבנת הנקרא ב-4 רמות** (מי / מה / למה / מה אתה חושב) + סדר אירועים + רגש + השלמת סוף.
- **הקלטה עצמית** — הילד שומע את עצמו ומשתפר. ההורה משווה ראשונה↔אחרונה.

### 🎮 מערכת ההתמכרות (gamification)
- **מפת הרפתקה 🗺️** — "מסע האור": שביל מתפתל מצויר עם תחנות (כמו משחק), נֵרִי מסמן "אתה כאן". **דילוג חופשי** — כל שלב פתוח.
- **משימה יומית 🎯** — 2 פעילויות + 20 אור = פרס. סיבה לחזור כל יום.
- **חנות האור 🛍️** — 23 פריטים: צבעי להבה, כובעים (כתר/כיפה…), חיות מחמד, רקעים. הדמות מתעדכנת מיד.
- **דרגות אוֹר ✨** — מ"ניצוץ" ועד "שמש" (7 דרגות).
- **מדליות 🏅** — 14 מדליות ארד/כסף/זהב + **לוח האור** (יומן רצף עם נרות).
- **אלבום הסיפורים 📔** — אוספים כרטיס לכל סיפור, נחשף עם הסיום, אפשר לקרוא שוב.
- **מגרש משחקים 🕹️** — משחקי ארקייד: צַיָּד הָאוֹתִיּוֹת (30 שניות), בּוּעוֹת מִלִּים.
- **תיבת אוצר יומית 🎁** — אור + לפעמים פריט מתנה.
- **קומבו 🔥** ("X ברצף!") · **נֵרִי משבח בשם הילד בקול** · **סצנות מצוירות לסיפורים**.
- **קריאה בעולם האמיתי 🌍** · **רגעי גאווה** (כרטיסים להדפסה) · **מרחב הורים** · **PWA אופליין**.

## 📂 מבנה
```
app/
  index.html          ← טוען הכל
  manifest.json · sw.js   ← PWA + אופליין
  css/style.css
  data/   letters.js · words.js · stories.js · curriculum.js   ← התוכן
  js/     state.js · audio.js · ui.js · activities.js · app.js  ← המנוע
  assets/ icon.svg
docs/   01-חזון-וחוויה.md · 02-החלטות-בנייה.md
```

## ➕ איך מוסיפים סיפור (בלי לגעת בקוד)
פותחים `app/data/stories.js` ומוסיפים אובייקט למערך — לפי אותו תבנית.
שדות חובה: `id, title, value, valueLabel, emoji, level, pages[], comprehension{who,what,why,think}`.
שדות רשות: `sequence[], emotion, ending`. הסיפור יופיע אוטומטית בעולם המתאים (level 4 = מנוקד, 5 = בלי ניקוד).
