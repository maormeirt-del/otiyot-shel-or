# פרק 2 — החלטות בנייה (Build Decisions)
### מערכת הקריאה החכמה · "אוֹתִיּוֹת שֶׁל אוֹר"

> ההחלטות שמכתיבות את הבנייה. כל אחת מהן נובעת מדרישה מפורשת של מאור.

---

## 1. אפס תלות · אפס עלות · אפס טוקנים
**דרישה:** "שלא יהיה תלוי בבינה מלאכותית כי אני לא רוצה להיות תלוי בטוקנים, כמו באפליקציית הבגרויות."

**מימוש:**
- אפליקציה **סטטית לחלוטין** — HTML/CSS/JS טהור, בלי שרת, בלי build, בלי npm, בלי API key.
- כל הסיפורים והתרגילים **כתובים מראש** בתוך האפליקציה (קבצי data).
- **הקראה ורמזים** = `speechSynthesis` — קול עברי שמובנה חינם בדפדפן/מערכת. לא OpenAI, לא תשלום.
- **הקלטה-עצמית** = `MediaRecorder` — מקומי במכשיר (IndexedDB). לא ענן, לא דליפת תוכן.
- **עלות תפעול: 0 ₪.** רץ מקובץ, או פריסה חינם ל-Vercel/GitHub Pages.

## 2. זיהוי-קול חכם = אופציה עתידית, לא תנאי
מנוע "זיהוי הניחוש"/"מנוע הביטחון" המלא דורש AI אמיתי. לכן:
- v1 **שלמה ומהנה בלעדיו.**
- "מנוע הביטחון" מיוצג ב-v1 דרך **הקלטה-עצמית** ("תשמע כמה השתפרת") — הפן הרגשי, בחינם.
- כשמאור ירצה — נוסיף שכבת זיהוי כ-opt-in (גם אז עדיף מנוע חינמי/מקומי על-פני טוקנים).

## 3. מותאם מגדר — בן / בת
**דרישה:** בחירה למי המשחק, בן או בת.
- בתחילת המשחק: בחירת **בן/בת** + שם + דמות.
- **כל** טקסט מותאם מגדר דרך הפונקציה `g(זכר, נקבה)` — פעלים, פניות, ברכות.

## 4. ציבור דתי · ערכים יהודיים-תורניים
**דרישה:** מתאים לציבור הדתי, ערכים יהודים תורניים.
- דמות מלווה: **נֵרִי** — להבה קטנה. מקור: "נֵר מִצְוָה וְתוֹרָה אוֹר" (משלי ו, כג).
- מטבע התגמול = **אוֹר** (לא מטבעות/יהלומים). הילד "מדליק אור" בכל הצלחה.
- כל סיפור נושא **מידה**: אמת, חסד, כיבוד הורים, שבת, צדקה, תפילה, ועוד.
- שפה, שמות וערכים מהעולם הדתי-לאומי.

## 5. פדגוגיה — סולם רכישת קריאה בעברית
6 רמות, מהאות עד הספר:
0. **אותיות** — צורה + שם + צליל
1. **ניקוד והברות** — בָּ בֵּ בִּ בֹּ בֻּ
2. **מילים קצרות מנוקדות**
3. **משפטים**
4. **סיפורים מנוקדים** + הבנת הנקרא
5. **סיפורים ללא ניקוד** (מתקדם)

## 6. ארכיטקטורה — PWA קודם
- Web/PWA טהור, מותקן, אופליין (Service Worker).
- בלי תלויות חיצוניות (גם לא ספריות JS) — הכול vanilla, נטען בלי אינטרנט.
- שלב הבא בעתיד: עטיפת Flutter לחנויות — אותו תוכן, בלי לזרוק כלום.

## 7. שם וזהות
